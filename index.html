Início

Dashboard de consulta rápida

Aviso de segurança: esta versão mobile preserva notas pessoais, rascunhos de consulta e pendências de revisão. Essas notas ficam somente no navegador, não são prontuário, não são criptografadas e não devem conter dados identificáveis de pacientes.
Instalar app: você pode adicionar este painel à tela inicial quando ele estiver aberto por um link HTTPS. No iPhone, use Safari → Compartilhar → Adicionar à Tela de Início.
